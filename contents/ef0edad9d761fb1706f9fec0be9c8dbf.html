Ordem de venda

UMLUseCase
Projeto 1 - Plataforma de Negocios :: Diagrama de Casos de Uso :: Ordem de venda

Description

  1. Usuário decide qual ação vender
  2. Usuário decide a quantidade a ser vendida (fracionário ou lote)
  3. Usuário escolhe o tipo de ordem (fluxo alternativo)
  4. Usuário envia OV para corretora
  5. Corretora envia OV para B3
  6. Corretora retorna a execução da ordem
  7. Usuário visualiza o status da OV
  8. Usuário se quiser pode modificar OV (fluxo alternativo)

3.1. Usuário usa/configura a ordem de venda com gatilho 3.2. Usuário determina uma condição a ser acionada (stop gain ou stop loss), com preço e valor do gatilho 3.3. Usuário insere a ordem de venda com gatilho

8.1. Usuário consulta o histórico de OV 8.2. Usuário seleciona a OV 8.3. Usuário modifica a OV 8.4. Usuário finaliza a modificação
