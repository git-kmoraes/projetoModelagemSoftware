Cadastro

UMLUseCase
Projeto 1 - Plataforma de Negocios :: Diagrama de Casos de Uso :: Cadastro

Description

  1. Usuário cria um cadastro, enviando documentação
  2. Liberação de cadastro após conferência da documentação
  3. Usuário faz login
  4. Usuário se necessário, pode acessar área de suporte (fluxo alternativo)
  5. Usuário acessa guia do iniciante para aprender as principais funcionalidades da plataforma (fluxo alternativo)

4.1. Usuário solicita recuperação de senha por e-mail 4.2. Usuário alterar senha pela área de suporte 4.3. Usuário faz login

5.1. Usuário assiste vídeo tutorial 5.2. Usuário finaliza o vídeo tutorial
